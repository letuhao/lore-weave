LoreWeave · Work Assistant · mobile · rev. 3

A journaling companion that lives in your thumb.

The desktop assistant is three columns — a sidebar, the chat, and a “today so far” rail. On a phone that layout can’t survive a squeeze, and shouldn’t. This is a redesign for the place the feature is actually used: one-handed, in a hallway between meetings, at the end of a long day. Chat becomes the whole screen; capture stays visibly transparent and pausable; the day’s one ritual — End my day — is always on the surface, never behind a gesture.

Rev. 3 covers the full feature set — 13 screens spanning the all-day loop, voice, recall, the journal timeline + entries, weekly reflection, coaching (practice → score), the memory you can inspect and forget, and the privacy controls. Every desktop surface now has a phone counterpart.

Revised after a 3-perspective review — low-tech busy user · accessibility/real-world · privacy-anxious

Low-tech & busy

  • “End my day” is now a visible button, not a hidden swipe-up
  • The today panel opens from a labelled bar (“3 noticed today ›”), not a mystery handle
  • Keep/skip say “Keep” / “Remove” — tap-first, swipe optional
  • Wording: “In review”→“Practice score”, “Open thread”→“To follow up”, one name (“Journal”)
  • Dropped the clingy “3h 12m together”

Accessibility & real-world

  • Every control is a real button/input with a label (screen-reader operable)
  • Tap targets raised to ≥44px (the 28px dismiss ✕, keep/skip, corners)
  • Light-theme muted text & the honesty badge darkened for AA in glare
  • State is never colour-only — active tab bolds + underlines; tags carry icons

Privacy & trust

  • Consent now defaults OFF (was wrongly shown ON — fail-closed)
  • Capture pill reframed “Held for tonight · nothing saved yet” + a reachable Pause
  • New “You & data” screen: see what’s remembered, forget a person, erase
  • Added the content-free lock-screen preview + a peek-protection toggle
