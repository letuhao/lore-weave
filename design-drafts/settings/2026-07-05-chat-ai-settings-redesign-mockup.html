LoreWeave · Settings redesign · Draft

Chat & AI — one place, nothing hidden, set once

Today the chat settings are scattered across seven surfaces, model choice is split three ways, TTS lives in two disconnected stores, and several behaviors (grounding, tool-write authority, reasoning, the TTS voice) are chosen silently with no control. This redesign folds all of it into one panel built on a single idea: your defaults live here, and every chat and every Writing Studio tool inherits them — override anywhere, never required.

Chat & AI · Models

Models

Shared with Writing Studio

Pick your models once here. The chat window and every Studio tool — co-writer, compose, planner, rewrite, scene-graph — start from these. You can still swap the model inside any tool for a single job, but you never have to.

How inheritance works

Each role below sets a default. A new chat session copies it; a Studio tool reads it live. An override you make in one place (a session, one tool) stays there and doesn't touch your default — close it and everything falls back to what's set here.

Chat & AI · Grounding & Memory

Grounding & Memory

What the assistant pulls from your story bible and knowledge graph before it answers. The biggest hidden default in the whole product lives here.

Grounding is always ON today with no way to turn it off. Every turn retrieves glossary, facts, and passages — governed only by a global engine flag, invisible to you. This panel gives it an actual switch (default stays On — it's what makes answers accurate).

Advanced · Context management

Context management

Auto

How the assistant manages a very long conversation's memory budget. These were process-global engine flags — the same for every user, invisible, unchangeable. Here they become a per-session setting that derives from your account default. Most people leave it on Auto.

Leave this on Auto

Auto turns the long-work tiers on only when a project is big enough to need them (thousands of chapters, a large glossary, or a conversation filling the model's window). The controls below matter for a handful of very large books — everyone else can ignore them.
