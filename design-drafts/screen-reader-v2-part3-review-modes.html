Mode 1 — Translation Review (Split-Pane)
Block-aligned split view: Source and translation blocks are paired by index. Each row shows the source text (left) and its translation (right). Media blocks show the same asset on both sides with translated captions. Status dots indicate translation state per block.

Uses ContentRenderer internally: Each side renders blocks via the same component, just with different content and a compact layout mode.
