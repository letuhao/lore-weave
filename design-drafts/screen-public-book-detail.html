Design Notes — Public Book Detail (MIG-07)

Layout

  • Breadcrumb: Browse > Book Title
  • Hero: Cover (200px) + Info (flex)
  • Chapter list: card with rows, pagination
  • Stats footer: 4 cards grid
  • Max-width 1000px, centered

Data Sources

  • GET /v1/catalog/books/:id (no auth)
  • GET /v1/catalog/books/:id/chapters (paginated)
  • Cover: book.cover_url or gradient fallback
  • Translations: from chapter list response
  • Click chapter → /read/:bookId/:chapterId

Actions

  • "Start Reading" → /read/:bookId/:firstChapterId
  • "Add to Library" → future (bookmarks)
  • "Share" → copy URL
  • Language selector changes chapter view language
  • Chapter row click → reader for that chapter

States

  • Loading: skeleton for cover + info + chapter list
  • Not found: 404 message with back to Browse link
  • No chapters: empty state message
  • Translation badge opacity: full = available, 50% = partial
