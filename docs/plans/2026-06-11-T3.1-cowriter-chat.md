## CoWriterActions logic
- latest = last message with `role==='assistant'` and non-empty content.
- if latest && !isStreaming → `[Insert as draft]`(→onInsert(latest)) `[Use as guide]`(→onUseAsGuide(latest)).
- else if `activeSession && messages.length===0` → starter chips → `send(prompt)`.
- else → nothing.

## CompositionPanel wiring
- `const [composeGuide, setComposeGuide] = useState('')` → `<ComposeView guide={composeGuide} onGuideChange={setComposeGuide} …/>`.
- `<CoWriterChat bookId onAccept={onAccept} onUseAsGuide={(t)=>{setComposeGuide(t); setTab('compose')}} />` (CSS-hidden always-mounted).

## Test outline (vitest)

- CompositionPanel: 15-panel mount; cowriter survives a tab toggle (visibility).
- ComposeView: existing tests pass with guide as a prop.
