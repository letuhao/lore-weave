## arcBandSplit (pure)
- `status==='gone'` & `from_order` set → split = index of first event with `event_order >= from_order` (band: active `[0,split)`, gone `[split,count)`).
- `status==='gone'` & `from_order==null` → split 0 (whole arc gone).
- else (active) → split = count (no gone band).
Band x via the same midpoint clamp as the cutoff markerX.

## Test outline (vitest)
- **pure** `arcBandSplit`: null→count; mid→correct index; before-all→0; gone-no-order→0.
- **component** (mock `useCharacterArc` + `useNavigate`): arc events ordered + spoiler cut + dim; relations strip (direction-aware, both sides); gone → active+gone rects, active → no gone rect; picker change → `onEntityChange`; click event → navigate; empty; noProject.
- **CompositionPanel**: 13-panel mount + arc launch (Cast 📈 → tab switch) if cheap to assert.
