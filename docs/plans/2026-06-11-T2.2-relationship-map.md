## GraphCanvas (generic, controlled) — `components/GraphCanvas.tsx`
Owns ONLY the shared SVG mechanics; everything consumer-specific is a prop.
```ts
type Pos = { x: number; y: number };
type GraphCanvasProps<E> = {
  positions: Record<string, Pos>;                 // controlled (consumer owns)
  nodeIds: string[];
  edges: E[];
  edgeEndpoints: (e: E) => { from: string; to: string };
  renderNode: (id: string, h: { onPointerDown: (e: React.PointerEvent) => void }) => React.ReactNode;
  renderEdge: (e: E, from: Pos, to: Pos) => React.ReactNode;
  nodeSize: { w: number; h: number };
  onNodeClick?: (id: string) => void;
  onNodeDrag: (id: string, pos: Pos) => void;     // live
  onNodeDragEnd?: (id: string, pos: Pos) => void; // commit/persist
  onBackgroundClick?: () => void;
  defs?: React.ReactNode;
  minWidth?: number; minHeight?: number;
  testid?: string;
};
```
Internals: drag ref + `startDrag(id)` (passed to renderNode as `onPointerDown`); svg-level `onPointerMove`/`onPointerUp` (5px threshold → drag→onNodeDrag/onNodeDragEnd, else→onNodeClick); `setPointerCapture` guarded; extent w/h from positions+nodeSize+min; transparent background rect → onBackgroundClick; `defs` slot; renders edges (endpoints→positions, skip missing) then nodes (renderNode). **SceneNode/SceneEdge stay unchanged** (they already take pos/onPointerDown/onSelect/onOpen).
