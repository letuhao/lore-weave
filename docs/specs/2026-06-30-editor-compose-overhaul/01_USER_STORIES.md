# 01 — User Stories

> Discussion is **story by story** with the PO. Each story's detailed discussion + locked decisions
> live in [`stories/`](stories/). Status here mirrors the [README tracker](README.md).

Persona: **the author** working in the chapter workspace (multi-tenant; their own books).

---

## Epic A — One coherent workspace with modes
- **A1.** See a single, obvious **Write / Translate / Compose** switch, so I always know my mode and
  how to change it. _(status: ⬜)_
- **A2.** Keep the same chapter, scenes, glossary, and AI context when I switch modes. _(status: ⬜)_

> **Manual "Write" mode** — definition & quality of the manual-writing baseline that underlies A1's
> "Write". 🟡 **discussing** → [`stories/01-manual-write.md`](stories/01-manual-write.md).

## Epic B — Translate as a real mode  🟡 discussing → [`stories/02-translation.md`](stories/02-translation.md)
- **B1.** Enter a persistent **Translation mode** for the open chapter (not a one-off button). _(⬜)_
- **B2.** See languages/coverage, pick or activate a version, start/track a translation job. _(⬜)_
- **B3.** Read source and translation side-by-side and edit a translated block. _(⬜)_
- **B4.** *(new)* **Author a translation by hand** (human-first, no LLM) — create a human version
  seeded from the source and edit it side-by-side. **Needs a small translation-service endpoint.** _(⬜)_

## Epic C — Compose I can actually drive
- **C1.** Open Compose onto a **start / command-center** that says what I can do + where I left off. _(⬜)_
- **C2.** Find the ~24 tools **grouped into a few named sections** so I navigate by intent. _(⬜)_
- **C3.** **Chat about my prologue/idea and then continue** — turn the discussion into a scene/draft
  via a visible Insert / Use-as-guide / Create-scene path. _(⬜)_
- **C4.** Not be confused by two different "Compose" features — each clearly named and placed. _(⬜)_

## Epic D — Scenes I can organize
- **D1.** **Add, reorder (drag), arrange** scenes from a first-class Scenes panel. _(⬜)_
- **D2.** **Archive / restore** scenes. _(⬜)_
- **D3.** Jump from a scene to its draft in one click. _(⬜)_
