- Pick a focus entity вҶ’ render it + its **1-hop relations** (`GET /entities/{focus}`). Each edge shows its `predicate` (on hover); each neighbor node is clickable to **re-focus** (load its 1-hop) вҖ” progressive graph exploration.
- **Expand** a node вҶ’ merge its 1-hop into the view (multi-hop by accretion) with a node cap.
- Node badge by `kind`; edge style by relation confidence / `pending_validation` (dashed = unconfirmed).
- Hand-rolled **SVG** (cross-cutting decision; no graph lib), auto-layout (force-ish / radial around focus) + per-device drag positions in localStorage.
- Click a node вҶ’ also offers "open in Cast codex" (T2.1).
- Empty (no relations) вҶ’ hint.

## 2. FE shape

- **hook** `features/composition/hooks/useRelationshipMap.ts` вҖ” focus state + a map of `entityId вҶ’ EntityDetail` (accreted); `useEntityDetail` per focus/expand (reuse `features/knowledge/api.ts`). A pure `buildGraph(details)` вҶ’ `{nodes, edges}` (dedupe nodes/edges) + `radialLayout(nodes, focus)` (unit-tested).
- **component** `features/composition/components/RelationshipMap.tsx` (view) вҖ” SVG nodes + edges; click-refocus, expand, drag. Extract `GraphNode`/`GraphEdge`; reuse the T1.3 SVG primitives if shared.
- **api / types** вҖ” reuse `features/knowledge` (`getEntityDetail`, `Entity`, `Relation`).
- **host** вҖ” **T5.5** power-view overlay (graph wants full width); add as a 4th overlay view. Build standalone first, graft at T5.5.

## 3. BE delta

**None for V1** (ego-network from existing 1-hop). **Optional follow-up:** a `GET /entities/{id}/graph?depth=N` would replace N client round-trips with one query вҖ” deferred (track as D-COMP-RELMAP-GRAPH-ENDPOINT) unless N-hop perf bites.

## 4. Data contract

`GET /v1/knowledge/entities/{entity_id}` вҶ’ `EntityDetail { entity: Entity, relations: Relation[] }`. `Relation { relation_id, subject, predicate, object, confidence, pending_validation, source_type, вҖҰ }`. The graph node set = focus + neighbors (subject/object of each relation); edges = relations.

## 5. Acceptance criteria

- Author picks a focus entity and sees its relations as a navigable graph.
- Clicking a neighbor re-focuses; expanding accretes that node's 1-hop (capped).
- Unconfirmed relations are visually distinct; hovering an edge shows the predicate.
- No relations вҶ’ a clear empty state.

## 6. Test outline

- **vitest**: `buildGraph` dedupes nodes/edges across accreted details; `radialLayout` deterministic; `RelationshipMap` вҖ” nodes/edges render; click вҶ’ refocus fetch; expand merges; unconfirmed edge style.
- **Playwright** `composition-relationship-map.spec.ts`: focus an entity вҶ’ neighbors render; click a neighbor вҶ’ re-focus.
- **i18n Г—4** keys: `relmap.title`, `relmap.focus`, `relmap.expand`, `relmap.unconfirmed`, `relmap.empty`.

## 7. Decisions (cleared 2026-06-10)

- **Ego-network V1** (accretion from 1-hop); a server-side N-hop graph endpoint is the scale path (deferred, D-COMP-RELMAP-GRAPH-ENDPOINT). (Lead)
- **Node cap ~60 + collapse** for density. (Lead)
- **Host:** T5.5 power-view overlay (a 4th overlay view). (Lead)
- **Shared `<GraphCanvas>` SVG primitive** with T1.3 (Scene Graph) + T2.5 (World Map) вҖ” design the three together. (Lead)
