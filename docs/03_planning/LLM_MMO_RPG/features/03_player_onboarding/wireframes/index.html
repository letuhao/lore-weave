PO_001 Player Onboarding — Wireframes

Static HTML mockup · Phase 0 FE design · Anchor: BG3 dual-mode + Disco Elysium amnesia + AI Dungeon freedom

Common pre-flow

00 — Landing / Login

Account creation + returning login. Email + password V1; OAuth V1+.

 01 — Reality Selection

Pick world (Đông Phong / Saigon 2026 / Faerûn). Per-reality theme switches.

Path choice (3 V1 modes)

02 — Path Choice

Pick (A) Canonical PC / (B) Custom PC / (C) Xuyên Không Arrival.
03A — Canonical PC

BG3 Origin pattern — pick from preset PCs. 5 NPCs + portraits + bios.

 03B — Custom PC (Basic Wizard)

8-step wizard — name → race → origin → personality → faction → appearance → spawn → review.

 03C — Xuyên Không Arrival

Disco Elysium amnesia + wuxia transmigration. Soul + Body + LeakagePolicy.

Power-user modes (NEW 2026-04-27)

Mode B "Custom PC" mở rộng với 2 power-user views: Advanced Settings (toàn bộ ~46 V1 fields) + AI Assistant (natural-language fill).

03B2 — Advanced Settings

Edit toàn bộ ~46 V1 actor settings across 14 features (Identity / Race+Origin / Personality+Mood+Beliefs / Knowledge / Family / Faction+Rep+Title / Skills / Resources / PC Body Memory). V1+ deferred fields shown as locked.

 06 — AI Character Assistant

Mô tả nhân vật bằng tiếng Việt → AI điền 18+ fields với reality-aware constraint checking + cross-feature consistency validation. Iterative tweak mode.

📖 Xem ACTOR_SETTINGS_AUDIT.md — comprehensive inventory tất cả 46 V1 actor settings + ~50 V1+ deferred.

Common post-flow

04 — Confirm & Bind

Summary card + LLM-generated opening preview. Forge:RegisterPc + BindPcUser cascade.

 05 — First Turn

Drop into spawn cell. SR11 turn UX visible from turn 1.

Reality theme preview

Reality-themed dynamic accent — visit landing with different ?reality= query param to preview themes.
