# LoreWeave

LoreWeave is a multi-agent platform concept for multilingual novel workflows:
- translation and terminology consistency,
- story knowledge construction (timeline, entities, relations, wiki-style outputs),
- grounded QA/extraction,
- canon-aware creative continuation.

## Current Repository State

This repository is currently in a **planning and governance phase**.
The primary artifacts are architecture, governance, and roadmap documents under `docs/`.
